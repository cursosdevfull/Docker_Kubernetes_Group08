# Imágenes

### Listar imágenes

```
docker images
```

### Inspeccionar una imagen

```
docker image inspect <nombre de imagen>
docker image inspect <nombre de imagen>:<tag de la imagen>
```

### Eliminar una imagen

```
docker rmi nginx:alpine
```

### Forzar la eliminación de una imagen

```
docker rmi -f nginx:alpine
```
